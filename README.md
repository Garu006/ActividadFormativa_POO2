Instrucciones

    Configuración del entorno
        Abre Android Studio.
        Crea un nuevo proyecto seleccionando la plantilla Empty Compose Activity.
        Verifica que el proyecto compile y ejecute correctamente en un emulador o dispositivo físico.
    Implementación de componentes básicos
        Agrega los siguientes componentes en tu interfaz:
            Text: para mostrar un título o mensaje.
            TextField: para ingresar información del usuario.
            Button: para ejecutar una acción.
            Image: para mostrar un recurso visual (puede ser local o desde internet).
        Asegúrate de que al menos dos componentes sean interactivos (por ejemplo: TextField y Button).
    Interactividad
        Programa el botón para que realice una acción, como:
            Mostrar el texto ingresado en pantalla.
            Cambiar el contenido de un Text.
        Maneja el estado utilizando remember y mutableStateOf.
    Personalización
        Aplica estilos básicos:
            Tamaño y color de texto.
            Espaciado entre componentes.
            Alineación simple usando Column o Row.
    Validación de usabilidad
        Presenta tu aplicación a un compañero o instructor.
        Solicita retroalimentación sobre:
            Claridad de la interfaz.
            Facilidad de uso.
            Comprensión de la funcionalidad.
    Ajustes finales
        Realiza mejoras basadas en la retroalimentación recibida.
        Asegúrate de que la interfaz sea clara, funcional y fácil de usar.

Entregable

    Código fuente del proyecto.
    Capturas de pantalla de la interfaz.
    Breve descripción (1–2 párrafos) sobre:
        Qué hace la aplicación.
        Qué cambios realizaste tras la validación.
